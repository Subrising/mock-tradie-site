# Dave's Fencing & Gates - Mock Tradie Website

A realistic Australian tradie website for testing QuoteShield widget integration.

## Purpose
This site is a mock business website created to test and demonstrate the QuoteShield lead qualification widget in a real-world context.

## QuoteShield Integration
The site includes the QuoteShield widget embedded in the contact section, allowing potential customers to get instant AI-powered estimates for fencing and gate jobs.

## View Live
Visit: https://subrising.github.io/mock-tradie-site/

## Local Development
Simply open `index.html` in a browser. No build process required.
